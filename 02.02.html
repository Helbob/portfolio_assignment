Assignment 13C.02.02 Designing with frameworks

In this assignment, I had to work with Bootstrap, Pico and a little bit of Tailwind. To learn how to use a CSS framework. This assignment I mostly used Bootsrap only.

Use of frameworks

I've mostly used Bootstrap to solve this assignment. I decided to go for their color "warning", which is the bright yellow color, that's seen throughout the whole page.

Motivation for the Final solution

For this assignment, I tried to find a default Bootstrap color, that would work well with the hero image of a forrest. So the theme for the website was forrest and then I added some foxes in as well. I tried to solve this assignment, in the same layout as the other assignments we've had, but with a few changes here and there.

I would've liked to use Tailwind instead of Bootstrap, as it has way more options for styling - Unfortunately there was not enough time for me to do so.
