Assignment 13C.02.01 Responsive design

In this assignment, I got to work with flex and grid. The meaning behind this assignment was to learn more about how to make a responsive website.

Responsive design

For this assignment, I decided to go with some simple colors and display the elements, as they were supposed to be placed. The website is of course also responsive if you go mobile. For this assignment I've used grid and flex.

Motivation for the Final solution

For this assignment I went with a simple color scheme and I also went with a wine farm aesthetic, which is why I chose the dark red color and the toned down yellow as a accent color. Before deciding on the colors that I went with, I did go over 3 other color schemes and ended up wit the wine farm aesthetic one.

The entire website is responsive in mobile if you want to scale it down. I tried to make use of the flex / grid tips that we were given in as many places as possible.
